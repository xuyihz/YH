## 数据格式

### ANSYS (APDL)

|   命令    |       说明       |
| :--: | :--: |
|    ET     |     单元类型     |
|     R     | 实常数 (截面积)  |
|  MP (EX)  |     弹性模量     |
| MP (PRXY) |      泊松比      |
| MP (DENS) |       密度       |
|     N     | 节点(编号及坐标) |
|    EN     |  单元(连接节点)  |
|     D     |       约束       |
| INISTATE  |    初应力 等     |



### MIDAS GEN (MGT)

|     命令     | 说明 |
| :----------: | :--: |
|    \*NODE    | 节点 |
|  \*ELEMENT   | 单元 |
| \*CONSTRAINT | 约束 |
